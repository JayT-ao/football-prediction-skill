# Skill: Football Match Prediction

## Overview

专业足球比赛预测系统，基于数据驱动的分析方法。适用于世界杯、欧洲杯、联赛等各类足球比赛预测。

**核心逻辑：** 数据采集 → 积分形势分析 → 实力对比 → 概率估算 → 价值评估 → 串关策略

**触发场景：**
- 用户询问足球比赛预测
- 用户询问比赛胜负/比分
- 用户询问投注建议
- 用户询问串关怎么买
- 任何涉及足球赛事分析的问题

## Step 1: Data Collection

### 1.1 赛程和赔率
**数据源：** ESPN赛程页面
**获取方式：** webfetch ESPN赛程URL

**获取内容：**
- 比赛日期、时间、场地
- 博彩赔率（胜负盘、让球盘、大小分）
- 赔率公司：DraftKings

### 1.2 球队基本面
**数据源：** ESPN球队页面

**获取内容：**
- 小组排名、积分、净胜球
- 首轮比分、进球/失球数
- 进球排行榜（射手、助攻）
- 伤病情况、停赛信息
- 近期战绩（热身赛、预选赛）

### 1.3 教练和战术信息
**数据源：** ESPN新闻文章

**获取内容：**
- 教练赛前发布会内容
- 战术安排、阵型变化
- 球员状态评价
- 伤病更新

## Step 2: Standings Analysis

### 核心问题
这场比赛对双方有多重要？

### 形势判断矩阵

| 情况 | 球队心态 | 比赛态度 | 对预测的影响 |
|------|----------|----------|--------------|
| 必须赢才能出线 | 压力大、急躁 | 全力进攻，可能暴露防守 | 可能犯错 |
| 已经基本出线 | 轻松 | 可能轮换主力保存体力 | 可能大意 |
| 基本出局 | 无压力 | 可能放水或为荣誉而战 | 不确定 |
| 争夺小组头名 | 有动力 | 会全力争胜 | 战意强 |
| 生死战（输球出局） | 拼命 | 破釜沉舟 | 战意极强 |

### 概率调整参考
| 形势 | 概率调整 |
|------|----------|
| 必须赢（压力大） | -3% - -5% |
| 已经出线（轮换） | -5% - -8% |
| 基本出局（无动力） | -3% - -5% |
| 争夺头名（战意强） | +3% - +5% |
| 生死战（拼命） | +5% - +8% |

## Step 4: Probability Estimation

### 4.1 计算博彩隐含概率

正赔率：概率 = 100 / (赔率 + 100)
负赔率：概率 = |赔率| / (|赔率| + 100)

示例：
- 德国 -190 → 190 / 290 = 66%
- 瑞典 +350 → 100 / 450 = 22%

### 4.2 综合调整概率

最终概率 = 博彩隐含概率 ± 调整值

调整因素：
+ 积分形势有利（+3%~+8%）
- 积分形势不利（-3%~+8%）
+ 伤病利好（+3%~+5%）
- 伤病利空（-3%~+5%）
+ 状态出色（+3%~+5%）
- 状态低迷（-3%~+5%）

### 4.3 估算各比分概率

| 对阵类型 | 常见比分 | 概率范围 |
|----------|----------|----------|
| 强队vs弱队 | 3-0, 2-0, 4-0, 3-1 | 15%-20% |
| 势均力敌 | 1-1, 2-1, 1-0, 2-2 | 10%-15% |
| 弱队vs强队 | 0-1, 0-2, 1-2, 0-3 | 12%-18% |

---

## Step 5: Value Assessment

### 期望值计算公式

EV = (实际概率 x 赔率回报) - 100%

示例：
- 瑞典胜 +350（赔率4.5）
- 实际概率 34%
- EV = (34% x 4.5) - 100% = +53%

### 价值等级划分

| EV值 | 等级 | 建议 |
|------|------|------|
| > +30% | 五星 | 强烈推荐 |
| +10%~+30% | 四星 | 推荐 |
| +5%~+10% | 三星 | 可以考虑 |
| 0%~+5% | 两星 | 一般 |
| < 0% | 不推荐 | 不推荐 |

---

## Step 6: Parlay Strategy

### 串关组合原则

1. 选择铁胆场次 — 胜率>70%
2. 避开陷阱场次 — 胜率<55%或积分形势复杂
3. 控制场数 — 2-3场最稳
4. 分散投注 — 多个组合

### 串关类型

| 类型 | 场数 | 赔率范围 | 风险 | 适用场景 |
|------|------|----------|------|----------|
| 稳健型 | 2场 | 1.5-2.5 | 低 | 新手、求稳 |
| 平衡型 | 3场 | 2.5-5.0 | 中 | 有经验、平衡风险 |
| 进取型 | 3场 | 5.0-15 | 高 | 老手、搏高赔 |
| 激进型 | 4场+ | 15+ | 很高 | 博冷、高风险 |

### 串关赔率计算

串关赔率 = 场次1赔率 x 场次2赔率 x ... x 场次N赔率

## Example

### 案例：荷兰 vs 瑞典

**Step 1 数据采集：**
- 赔率：荷兰-135, 瑞典+350
- 瑞士首轮5-1胜突尼斯，荷兰2-2平日本
- 德容存疑，廷伯伤退

**Step 2 积分形势：**
- 荷兰1分必须赢，瑞典3分心态放松
- 荷兰压力大，瑞典有利

**Step 3 实力对比：**
- 瑞士5球(第1)+4助攻(第2)
- 荷兰2失球(第28)

**Step 4 概率估算：**
- 荷兰38%，平局28%，瑞典34%

**Step 5 价值评估：**
- 瑞典胜EV +53%，高价值

**Step 6 最终输出：**
- 首选：1-2瑞典(14%)
- 推荐：瑞典胜+350
- 串关：瑞典胜+厄瓜多尔胜(赔率5.06)

## Important Notes

| 事项 | 说明 |
|------|------|
| 数据时效性 | 赛前伤停变化会影响预测，赛前1小时再确认 |
| 足球随机性 | 即使99%概率也可能翻车，做好心理准备 |
| 博彩抽水 | 长期来看庄家必赢，不要沉迷 |
| 量力而行 | 只用闲钱，别影响生活 |
| 不要追损 | 输了就停，保持理性 |
| 记录复盘 | 每次预测后记录结果，不断优化流程 |
