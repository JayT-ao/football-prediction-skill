# Football Prediction Improvements

## 复盘发现的问题

### 1. 进球数预测偏差
- 预测4场总进球：11球
- 实际4场总进球：7球
- 偏差：+57%（严重高估）

### 2. 胜负方向准确率
- 准确率：50%（2/4）
- 错误：低估摩洛哥、高估土耳其

### 3. 平局概率偏低
- 实际平局率高于预测
- 需要提高平局基准概率

### 4. 心理因素利用不足
- 必须赢不等于一定赢
- 压力可能导致发挥失常

---

## 改进方案

### 改进1：泊松分布进球预测

原理：假设进球服从泊松分布，基于两队进攻防守能力计算各比分概率。

公式：P(X=k) = (lambda^k * e^(-lambda)) / k!

核心函数：

predict_score_probabilities(lambda_a, lambda_b, max_goals=5)
- 输入：A队预期进球、B队预期进球
- 输出：各比分概率字典

calculate_expected_goals(team_a, team_b)
- 输入：两队数据
- 输出：(lambda_a, lambda_b)

预期进球计算方法：

| 指标 | 计算方式 |
|------|----------|
| 进攻能力 | 场均进球 + xG |
| 防守弱点 | 场均失球 + xGA |
| 主场调整 | 主场x1.15 / 客场x0.85 |
| 状态调整 | 正负0.2（近期状态） |

---

### 改进2：平局概率调整

基准概率：27%（世界杯小组赛平均）

调整因素：

| 因素 | 调整值 | 说明 |
|------|--------|------|
| 势均力敌 | +5% | 实力差距小于10% |
| 双方保守 | +8% | 都以防守为主 |
| 一方必须赢 | -5% | 会冒险进攻 |
| 恶劣天气 | +5% | 大雨高温等 |
| 中立场地 | +3% | 无主场优势 |
| 小组末轮 | +5% | 双方都谨慎 |

核心函数：

calculate_draw_probability(base_prob=27, factors=None)
- 输入：基准概率、调整因素
- 输出：调整后的平局概率

---

### 改进3：心理因素评估

心理因素分类：

| 因素 | 影响 | 调整值 |
|------|------|--------|
| 必须赢（强队） | 可能超常发挥 | +3% |
| 必须赢（弱队） | 可能发挥失常 | -3% |
| 已经出线 | 轮换大意 | -5% |
| 基本出局 | 无压力 | +2% |
| 争夺头名 | 战意强 | +3% |
| 生死战 | 破釜沉舟 | +5% |

核心函数：

psychological_adjustment(team, situation)
- 输入：球队信息、比赛形势
- 输出：调整值

---

### 改进4：冷门独立评估

冷门概率计算逻辑：

1. 计算实力差距
2. 评估近期状态
3. 参考历史交锋
4. 综合计算冷门概率

核心函数：

calculate_upset_probability(favorite, underdog)
- 输入：强队数据、弱队数据
- 输出：冷门概率（弱队获胜概率）

---

### 改进5：综合概率计算

完整预测流程：

1. 计算预期进球
2. 计算各比分概率
3. 计算胜负概率
4. 调整平局概率
5. 重新分配概率
6. 计算冷门概率
7. 整理结果

核心函数：

comprehensive_prediction(team_a, team_b, match_info)
- 输入：两队数据、比赛信息
- 输出：完整预测结果

## 使用指南

### 步骤1：收集数据

球队数据格式：
- name: 球队名称
- rating: FIFA评分
- attack: 场均进球
- defense_weakness: 场均失球
- form: 近期状态（1-10）
- xG: 预期进球
- xGA: 预期失球

### 步骤2：分析比赛形势

比赛信息格式：
- must_win: 是否必须赢
- qualified: 是否已经出线
- eliminated: 是否基本出局
- neutral_venue: 是否中立场地
- bad_weather: 是否恶劣天气
- final_group_match: 是否小组末轮

### 步骤3：运行预测

调用 comprehensive_prediction 函数

### 步骤4：解读结果

- 预期进球：两队预期进球数
- 胜平负概率：各结果概率
- 最可能比分：前5个最可能比分
- 冷门概率：弱队获胜概率

---

## 注意事项

### 1. 数据质量
- 使用可靠的xG数据源（Understat, StatsBomb）
- 确保数据是最新的
- 考虑数据样本量（至少5场比赛）

### 2. 模型局限性
- 泊松分布假设进球独立（实际有相关性）
- 无法预测红牌点球等突发事件
- 心理因素难以量化

### 3. 持续优化
- 记录每次预测的结果
- 定期复盘调整参数
- 关注新的数据源和模型
