# Advanced Football Prediction Techniques

## 高级统计指标详解

### xG（Expected Goals，预期进球）

定义：基于射门位置、角度、防守压力等因素计算的进球概率。

数据来源：
- Understat（免费，欧洲五大联赛）
- StatsBomb（付费，顶级数据）
- FBref（免费，综合性数据）

使用方法：
1. 比较两队xG，判断谁创造机会更多
2. 比较xG与实际进球，判断运气成分
3. xG差值越大，说明比赛结果越合理

### xA（Expected Assists，预期助攻）

定义：基于传球位置、类型、防守压力等因素计算的助攻概率。

用途：
- 评估球员创造力
- 找出被低估的中场球员

### PPDA（Passes Per Defensive Action）

定义：对手在己方半场的传球数除以己方防守动作数。

含义：
- PPDA越低，逼抢越凶
- PPDA越高，阵地防守为主

## 博彩公司分析模型

### 赔率背后的信息

赔率变化含义：
- 赔率下降：有资金投入，看好该选项
- 赔率上升：资金流出，不看好该选项
- 剧烈变化：重大消息（如伤病）

如何读懂赔率：
如果我们的概率大于赔率隐含概率，有价值
如果我们的概率小于赔率隐含概率，无价值

## 高级概率模型

### 埃洛评分系统（Elo Rating）

定义：基于历史战绩的动态评分系统。

计算公式：
新评分 = 旧评分 + K * (实际结果 - 预期结果)
K = 20（世界杯）
预期结果 = 1 / (1 + 10^((对手评分 - 己方评分) / 400))

### 泊松分布模型

定义：假设进球服从泊松分布，计算各比分概率。

计算公式：
P(X = k) = (lambda^k * e^(-lambda)) / k!
lambda = 预期进球数
k = 实际进球数

## 高级串关策略

### 1. 对冲串关（Hedging Parlay）

原理：当串关前几场都正确时，通过反向投注锁定利润。

### 2. 系统投注（System Betting）

原理：从N场比赛中选择M场进行组合。

组合数 = C(N, M)
如果选5场买3串1，组合数 = 10种

### 3. 阶梯串关（Progressive Parlay）

原理：逐步增加串关场数，前一场中了才继续。

## 风险管理高级技巧

### 1. 凯利公式进阶

标准凯利：
f = (bp - q) / b
f = 最优投注比例
b = 赔率 - 1
p = 胜率
q = 1 - p

半凯利：
f_half = f / 2
优点：降低波动，减少破产风险

1/4凯利：
f_quarter = f / 4
优点：极低波动，适合新手

### 2. 动态资金管理

递增策略：
赢了增加投注（+10%）
输了减少投注（-10%）

递减策略：
赢了减少投注（-10%）
输了增加投注（+10%）

### 3. 分散投注策略

多串关组合：
组合1：稳健型（3场）- 投60%
组合2：平衡型（3场）- 投30%
组合3：进取型（3场）- 投10%

## 常见错误避免

### 1. 赌徒谬误
错误：已经连输5场，下场一定赢
正确：每场比赛独立，不受历史影响

### 2. 确认偏误
错误：只找支持自己观点的数据
正确：客观分析所有数据

### 3. 过度自信
错误：我是专家，我不会错
正确：承认不确定性

### 4. 追损
错误：输了就加倍投注
正确：冷静分析，总结教训

### 5. 忽视抽水
错误：赔率50%就是50%概率
正确：实际概率低于50%

### 常用公式

隐含概率 = 100 / (赔率 + 100) 正赔率
隐含概率 = |赔率| / (|赔率| + 100) 负赔率

EV = (实际概率 * 赔率回报) - 1

串关赔率 = 赔率1 * 赔率2 * ... * 赔率N

串关胜率 = 胜率1 * 胜率2 * ... * 赔率N

凯利比例 = (赔率 * 胜率 - 1) / (赔率 - 1)
