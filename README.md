# Football Match Prediction Skill

专业足球比赛预测系统，基于数据驱动的分析方法。

## Features

- 数据驱动：基于ESPN实时数据、博彩赔率、球队统计
- 积分形势分析：考虑出线形势对比赛态度的影响
- 高级统计：xG、xA、PPDA等专业指标
- 概率估算：泊松分布、埃洛评分等数学模型
- 价值评估：计算EV值、凯利公式优化投注
- 串关策略：稳健/平衡/进取型方案
- 风险管理：资金管理、止损策略、心态控制
- 心理因素：考虑球队心理素质和比赛压力
- 冷门评估：独立评估冷门概率
- 天气因素：影响比赛节奏和进球数
- 裁判因素：影响比赛强度和判罚
- 历史交锋：提供心理优势参考
- 主客场因素：更细致的主客场分析
- 球队疲劳度：影响球员体能和表现
- 战术风格：不同风格的克制关系
- 球星状态：关键球员的影响
- 球队士气：影响场上表现
- 小组赛形势：影响球队策略
- 让球盘口：市场预期的参考

## Advanced Features

### 核心统计指标
- xG（Expected Goals）：预期进球
- xA（Expected Assists）：预期助攻
- PPDA：逼抢强度

### 概率模型
- 泊松分布：比分概率计算
- 埃洛评分：球队实力评估
- 布拉德利-特里模型：胜负概率

### 心理因素评估
- 必须赢的压力影响
- 已经出线的轮换风险
- 基本出局的无压力状态
- 生死战的破釜沉舟

### 冷门独立评估
- 实力差距分析
- 近期状态对比
- 历史交锋参考
- 综合冷门概率

### V2新增因素

#### 天气因素
- 大雨：进球-15%, 控球-20%
- 高温：进球-10%, 体能-20%
- 大风：进球-10%, 控球-15%

#### 裁判因素
- 严格型：抑制对抗, 减少进球
- 宽松型：比赛流畅, 可能多进球
- 点球倾向：增加进球机会

#### 历史交锋
- 近5次交手权重40%
- 主客场战绩权重30%
- 大赛交手权重20%

#### 球队疲劳度
- 7天内比赛场次
- 上一场比赛强度
- 旅行距离
- 关键球员出场时间

#### 战术风格匹配
- 控球型 vs 反击型
- 控球型 vs 高压型
- 反击型 vs 防阵型

#### 球星状态
- 近期进球/助攻
- 伤病情况
- 心理状态

#### 球队士气
- 近期战绩影响
- 更衣室氛围
- 教练信心

#### 小组赛形势
- 必须大胜：+20%
- 必须赢：+10%
- 争头名：0%
- 保出线：-10%
- 已出局：-20%

#### 让球盘口
- -0.5：+3%
- -1：+5%
- -1.5：+8%
- 0：0%
- +0.5：-3%
- +1：-5%

## Risk Management

### 凯利公式
f = (bp - q) / b
建议使用半凯利或1/4凯利降低风险

### 资金管理
- 只用闲钱
- 设定预算
- 不要追损
- 分散投注

## Value Assessment

EV = (实际概率 x 赔率回报) - 100%

| EV值 | 建议 |
|------|------|
| 大于+30% | 强烈推荐 |
| +10%到+30% | 推荐 |
| +5%到+10% | 可以考虑 |
| 0%到+5% | 一般 |
| 小于0% | 不推荐 |
